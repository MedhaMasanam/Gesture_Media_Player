# Gesture Media Player

A computer vision application that allows users to control media playback using hand gestures via webcam input. Built as a college mini-project to explore real-time gesture recognition.

## 🚀 Features
- Play/Pause media using hand gestures
- Volume control using distance between fingers
- Real-time webcam processing

## 🛠️ Tech Stack
- **Python**
- **OpenCV** (for webcam processing)
- **MediaPipe** (for hand landmark tracking)
